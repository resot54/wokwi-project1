# Проект Wokwi

Схема, созданная в онлайн‑симуляторе Wokwi.

## Описание

Схема для генерации случайного числа Фибоначчи с выводом на LCD.

## Ссылка на проект Wokwi

https://wokwi.com/projects/461531618393817089

* `main.cpp` — код логики работы схемы.
